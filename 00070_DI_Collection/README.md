Spring examples to show you how to inject values into collections type (List, Set, Map, and Properties). 4 major collection types are supported :

List 每 <list/>
Set 每 <set/>
Map 每 <map/>
Properties 每 <props/>


## output of the Project

CustomerCollection [
list=[1, Person [name=xing hua, address=fjewoi rd, age=20], Person [name=Alice, address=alice rd, age=10]], 
set=[1, Person [name=xing hua, address=fjewoi rd, age=20], Person [name=Alice, address=alice rd, age=10]], 
map={1=1, 2=Person [name=xing hua, address=fjewoi rd, age=20], 3=Person [name=Alice, address=alice rd, age=10]}, 
pro={admin=admin@nospam.com, support=support@nospam.com}
]
